# 🌐 Personal Portfolio

A modern, responsive portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, certifications, and learning journey.

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript

## ✨ Features

- Responsive design
- Smooth animations
- About section
- Skills showcase
- Projects gallery
- Certificates section
- Contact section
- Social media links

## 📂 Project Structure

```
portfolio/
├── index.html
├── style.css
├── script.js
├── assets/
└── README.md
```

## 🛠️ Getting Started

Clone the repository:

```bash
git clone https://github.com/atul247ui/portfolio.git
```

Open `index.html` in your browser or use **Live Server** in VS Code.
